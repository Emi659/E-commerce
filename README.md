# Shopix Full-Stack E-Commerce 
Plataforma de comercio electrónico desarrolada con **Python (Flask)**,
que implementa una arquitectura moderna para la navegación, gestión de categorías
y experiencia de usuario optimizada



## Caracteristicas del sistema 
Inicio de sesion y/o dar de alta a un usuario: El sistema cuenta con un apartado donde el usuario se podra logear
o donde podra crear una cuenta nueva
